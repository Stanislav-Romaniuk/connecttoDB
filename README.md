# Connect to database
Simple Node.js example that demonstrates how to connect to a PostgreSQL database using the pg library and environment variables via dotenv.
---
# Start
## nmp install
``` npm install ```
## If starting from scratch, install required dependencies:
``` npm install pg dotenv ```

# Clone the project
``` git clone https://github.com/Stanislav-Romaniuk/connecttoDB.git ```
## go to the project folder
```cd connecttodb ```

# Launch
``` node db.js ```
